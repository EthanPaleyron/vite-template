## Use of [Vite](https://vitejs.dev/), [PNPM](https://pnpm.io) and [SASS](https://sass-lang.com/).

## Install the software :

- Install Vite : [NodeJS](https://nodejs.org/en).

## Project launch :

- Open your `Terminal`.

- Install dependencies in the project.

```bash
pnpm i
```

- Run the development.

```bash
pnpm run dev
```
